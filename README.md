# Lengux （使用bootstript 构建的一款 Hexo 主题 非常简洁）

## 💻 Installation

```
git clone https://github.com/XUSHAO11/Lengux.git
```

- node>=12
- 进入主题themes 使用git安装
- 修改hexo根目录 _config.yml
```
theme: Lengux
prismjs:
  enable: true #代码高亮，不打开的话，文章代码片段会出错
```
### 根目录安装依赖

```
npm i hexo-renderer-sass
npm i hexo-generator-search
```
### 文章标题配置

```
---
title: "文章标题"
intro: "文章副标题"
img: "文章封面"
article: "文章标签"
---
```
> theme: Lengux
